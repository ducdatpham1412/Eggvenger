# 🥚 Eggvenger – A Cute FPS Game Inspired by Valorant

**Eggvenger** is a first-person shooter game where players transform into adorable egg characters, each equipped with unique skills and guns. Inspired by the tactical gameplay of Valorant, the game combines shooting, strategic skill usage, and visibility-based mechanics for an engaging multiplayer experience.

## 🎮 Features

- **Cute Egg Characters**  
  Players take on the role of customizable egg heroes with personality and unique designs.

- **FPS Shooting and Skill System**

  - Equip **2 guns** and **3 skills** per player
  - Smooth gamepad-based controls: movement, aiming, shooting, skill casting, gun switching

- **2D Map Design**

  - Designed using Unity's **Tilemap** system
  - Optimized layout for tactical combat

- **Custom Visual Effects**

  - Skills and effects created with **Shader Graph** and **Particle System**
  - Emphasis on clarity, feedback, and charm

- **Advanced Vision System**

  - **Limited visibility** via a combination of:
    - **Light & Shadow mechanics**
    - **Custom rendering** using **Stencil Buffer**
    - **Line-of-sight** locked by terrain
  - **Mini-map** showing revealed areas

- **Multiplayer Mode**

  - Developed using **Unity Netcode for GameObjects**
  - Supports synchronized gameplay between players in the same match

- **Optimized Performance**
  - Implemented **Object Pooling** for bullets, VFX, and temporary game objects

## 🛠 Tech Stack

- **Engine**: Unity
- **Language**: C#
- **Networking**: Unity Netcode for GameObjects
- **VFX**: Shader Graph, Particle System
- **Input**: Gamepad Support
- **Rendering**: Stencil Buffer, 2D Lights & Shadows
